# Ricardo Personal

Site do Ricardo Personal — Educação Física, Personal Trainer. Treinos com método, disciplina e acompanhamento.

## Rodar o projeto

```bash
npm install
npm run dev
```

Acesse [http://localhost:5173](http://localhost:5173).

## Build para produção

```bash
npm run build
```

Os arquivos ficam na pasta `dist/`.
